Here is a quickstart guide to get you up and running with Chatter Platform. This will cover the steps required
to run your first prompt on Chatter. You can also visit the [wiki](https://anishagrawal.notion.site/Chatter-Wiki-31a4551c52784f1fbf4b7368f573fd9e?pvs=4) for video content.

<Steps>
    <Step title="Authenticate">
        Visit Chatter Platform and login to your account. If you don't have an account, you can create one for free.
        More details [here](/chatterplatform/authentication).
    </Step>
    <Step title="Vault">
        Create a vault in your workspace. This allows you to set which models are available to your workspace.
        More details [here](/chatterplatform/vault).
    </Step>
    <Step title="Create a collection / chain">
        Navigate to the sidebar in a workspace. Create a collection via the `+` button at the top of the sidebar (next
        to the workspace name). Then, create a chain within that collection via the `+` button next to the collection name. More details [here](/chatterplatform/chain).
    </Step>
    <Step title="Setup a prompt">
        Create a cell in your chain and write a prompt. Configure your LLM and paramters. For example try `Who was the {{ rank }} president of the {{ country }}` More details [here](/chatterplatform/cell).
    </Step>
    <Step title="Test">
        Open the testing suite and create a test case. Assign values to variables in the prompt and add assertions (expected response). More details [here](/chatterplatform/test-suite).
    </Step>
    <Step title="Run">
        Run your chain and see the results! More details [here](/chatterplatform/test-suite#running-a-chain-in-the-testing-suite).
    </Step>
</Steps>
